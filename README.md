# Lavie Learning

Static Hebrew soccer-themed learning game for practicing normal distribution concepts.

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

The product UI is fully Hebrew and currently uses soccer scenarios inspired by Champions League and Messi/Inter Miami references.

## Deploy to GitHub Pages

The app is static and has no build step. GitHub Pages can serve it directly from the repository root on `main`.
